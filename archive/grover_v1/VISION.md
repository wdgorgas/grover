# VISION

GROVER is a private AI operating layer for Will and Jackson: a server-hosted
command center with a desktop-quality cockpit, a living particle-orb identity,
a cost-aware model/tool router, structured long-term memory, loop-based agent
execution, and a Builder module that helps create the rest.

## The mindset

> Build the professional Grover kernel first: the system that can later help
> build and manage the rest.

Not a toy, not a generic chatbot, not every module at once. The kernel (this
repo, v1) carries: identity + memory separation, chat with routing and
budgets, the ledger, skills, audit, and the orb. Everything else — Research
Desk, Income Lab, Health, Media/Home, Inspiration Inbox, Morning Briefing,
Voice/Hardware — arrives as modules on top of these primitives.

## Non-negotiables

- **Loops over prompts**: serious workflows become bounded loops with budgets,
  verification, stop conditions, and memory of what happened.
- **Exoskeleton, not replacement**: Grover strengthens Will's thinking —
  organizes, retrieves, critiques, formalizes — and honestly labels output
  L0–L5. The Relativity Test gates every novelty claim.
- **Cost-aware by default**: cheap tier unless a task earns escalation;
  the dashboard is the truth.
- **Memory with boundaries**: private never crosses users; the vault stays
  human-readable forever.
- **Secure by architecture**: local-only bind, Cloudflare Access at the door,
  audit everywhere, external content is data not authority.
- **Beautiful restraint**: a quiet, professional interface where the orb is
  the one living thing — Arcane color richness, not HUD clutter.

## Module roadmap (in ledger, greenlit one at a time)

Builder (grover-dev slice of the ledger — active) → Loop Engine → Agent
Teams → GitHub integration → Research Desk → Income Lab → Inspiration Inbox →
Morning Briefing → Health → Media/Home → Voice/Hardware (dedicated hardware
preferred; never Siri-mediated by default).
